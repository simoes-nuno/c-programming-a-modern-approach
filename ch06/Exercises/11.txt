What output does the following program fragment produce?

sum = 0;
for (i = 0; i < 10; i++) {
    if (i % 2)              /* equivalent to if ( i % 2 != 0) */
        continue;
    sum += i;
}
printf("%d\n", sum);

This program adds up all the *even* numbers from 2 to 8 and prints the result.
2 + 4 + 6 + 8 = 20
